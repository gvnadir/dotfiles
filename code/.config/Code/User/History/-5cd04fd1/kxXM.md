# Python The Practical Guide

## Blockchain

Bitcoin is a cryptocurrency and criptocurrencies are based on the Blockchain technology.

A Blockchain is a distributed data storage consisting of containers (blocks) which are connected.

A single block of the Blockchain can hold any data you want (some text, a number, a program or a list of transaction).

The blocks in the Blockchain know about each other and they have an assigned hash. You got all these blocks and the next block always know about the previous block (the block cannot be edited withou the next block recognizing it).

## Criptocurrency

If the data you store in a block is a list of transactions (a piece of information saying Who is sending How Much to Whom), the coins transferred in the transaction form your criptocurrency.

## Coins

Coins are just what is transferred with a Transaction.

Coins are created via Mining. When we want to send something to someone, we will open a transaction and for that transaction to be confirmed it needs to be included in a new block which is added to the end of the Blockchain. People who are adding this are doing the so-called Mining and they get a reward out of it.

## REPL

Read, Evaluate, Print and Loop. REPL is an interactive shell where we can just enter commands and let Python do the job.
